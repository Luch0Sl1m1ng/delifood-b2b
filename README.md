# Delifood B2B - Gestión de Pedidos

MVP para la gestión automatizada de pedidos de materias primas gourmet.

## Tecnologías
* **Backend:** Node.js + Express
* **Frontend:** HTML5 + JavaScript + Bootstrap 5
* **Persistencia:** Memoria RAM (Arreglos)
* **Arquitectura:** Desacoplada (Frontend/Backend independientes)

## Funcionalidades
- [x] Login para Cliente y Administrador.
- [x] Creación de pedidos por parte del cliente.
- [x] Visualización global y cambio de estado por el administrador.

## Ejecución
1. Backend: `node backend/server.js`
2. Frontend: Abrir `frontend/index.html`
